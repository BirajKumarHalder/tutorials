Setup We require the following: • Nodejs • Npm • Angular CLI • IDE (VS Code) Download and install NodeJs To install Angular/CLI run “npm install @angular/cli –global” To create angular application run “ng new ” To run the application run “ng serve” The web server starts on port 4200. Type the url, http://localhost:4200/ in the browser and see the output. You can change the port if you wish using the following command: “ng serve --host 0.0.0.0 --port 4205”.
